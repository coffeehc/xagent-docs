# Version Comparison Rules

## Compare Structure First

Identify:

- document versions and dates;
- whether the same base document is being compared;
- sections added, removed, renamed, or moved;
- changed definitions, schedules, exhibits, attachments, and order forms;
- missing pages, truncated text, or OCR/copy errors.

## Change Types

Classify changes:

- new obligation;
- removed obligation;
- stricter requirement;
- relaxed requirement;
- changed deadline, notice period, renewal, cure period, or termination trigger;
- changed payment, liability, warranty, indemnity, privacy, security, IP, confidentiality, dispute, or assignment term;
- ambiguous wording change;
- non-substantive formatting or numbering change.

## Impact Notes

For material changes, state:

- what changed;
- who is affected;
- operational, financial, legal-review, privacy, security, or procurement implication;
- whether a reviewer should accept, reject, clarify, or escalate;
- what fact is missing if the impact cannot be determined.
