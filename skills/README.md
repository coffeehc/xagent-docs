# xAgent Built-in Skill Copies

This directory contains editable copies of selected xAgent built-in Skill packages that are suitable for user review and contribution.

These files are provided so users and contributors can inspect, discuss, and propose changes to built-in Skills in the docs repository.

Notes:

- This directory is a documentation/review copy, not the runtime installation directory.
- Changes here do not automatically change a running xAgent binary.
- Keep each Skill package complete: `SKILL.md`, referenced `references/`, `templates/`, `examples/`, `scripts/`, `assets/`, and license files should stay together.
- Do not add real secrets, customer data, private tokens, internal URLs, or one-off personal material.
- When changing a Skill, update `meta.version` in `SKILL.md` if the change is intended to become a new runtime version.

```text
docs/user-guide/builtin-skills.md
```
