# Sample Tasks

## 技术选型

1. 深度研究 PostgreSQL、MongoDB、DynamoDB 在多租户 SaaS 架构中的适用性，并给出选型建议。
2. 调研 LangGraph、OpenAI Agents SDK、CrewAI、AutoGen 的架构差异、生态成熟度和生产风险。
3. 研究向量数据库在企业 RAG 系统中的选型：Milvus、Weaviate、Pinecone、pgvector、Elasticsearch。

## 市场研究

1. 研究中国企业级 AI Agent 平台市场格局，包括主要玩家、商业模式、客户类型和机会。
2. 研究全球云安全态势管理市场，给出 2026 年进入策略。

## 竞品分析

1. 深度分析 Cursor、Windsurf、GitHub Copilot、Claude Code、Codex 在 AI coding agent 方向的差异。
2. 调研主流 API Gateway 产品在多租户、插件、安全和可观测性方面的能力。

## 政策法规

1. 研究欧盟 AI Act 对企业 AI 应用开发和部署的影响。
2. 调研中国数据出境相关法规对跨境 SaaS 架构的影响。

## 文献综述

1. 研究 2024–2026 年 multi-agent systems 在软件工程中的论文进展。
2. 综述 RAG evaluation 方法、指标、数据集和工具。
