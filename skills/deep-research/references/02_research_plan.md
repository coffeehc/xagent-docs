## 搜索轮次建议

### Round 1: Broad mapping

目标：建立主题地图，识别关键实体、术语、争议点、主要来源。

### Round 2: Targeted evidence

目标：围绕 checklist 的高优先级问题寻找权威证据。

### Round 3: Primary source acquisition

目标：获取官方文件、论文、标准、财报、监管文件、数据集、产品文档。

### Round 4: Contradiction and negative evidence

目标：寻找反例、批评、失败案例、限制条件、不同口径数据。

### Round 5: Freshness and final verification

目标：检查是否有最新版本、最新政策、最新价格、最新产品能力、最新数据。

## 查询角度

至少考虑：

- official / docs / standard / specification
- research paper / arxiv / conference / journal
- market size / report / filing / investor presentation
- benchmark / performance / comparison
- limitation / risk / critique / failure / incident
- pricing / release notes / changelog
- regulation / compliance / policy
- case study / adoption / migration

## 进入下一阶段条件

不要机械执行固定轮数。根据 gap matrix 决定是否继续。
