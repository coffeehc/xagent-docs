# xAgent 任务：目标、材料与验收标准

## 适用对象

本文适合希望让 xAgent 更稳定完成工作的普通用户。

## 这是什么

任务就是你希望 xAgent 帮你完成的一件事。它可以是一句简单要求，也可以是一个需要文件、工具、确认和多轮沟通的复杂工作。

使用 xAgent 时，不需要先学习模型、工具或内部概念。你只需要把“要做什么、材料在哪里、结果要什么样”说清楚。

## 什么时候使用

只要你希望 xAgent 做一件可以判断是否完成的事，就可以提交任务。例如：

- 阅读文件并总结。
- 整理会议记录。
- 生成报告、表格或清单。
- 检查资料中的问题。
- 根据外部消息起草回复。
- 定时执行固定工作。

## 提交前检查

提交任务前，先确认四件事：

| 检查项 | 说明 |
| --- | --- |
| 目标 | 希望 xAgent 最后交付什么 |
| 材料 | 文件、链接、背景、数据是否已经提供 |
| 范围 | 哪些内容要处理，哪些内容不要处理 |
| 结果 | 要直接回复、保存文件、生成表格还是生成报告 |

如果任务涉及发送、删除、访问外部系统，建议额外写一句“先给我确认，不要直接执行”。

## 基本写法

推荐按下面结构写任务：

```text
请基于【材料】，完成【目标】。
要求：
1. 处理范围是【范围】。
2. 输出格式是【格式】。
3. 如需发送、删除或调用外部系统，先让我确认。
```

示例：

```text
请阅读我刚上传的客户访谈记录，整理客户提到的主要问题和机会点。
要求：
1. 按“问题 / 证据 / 影响 / 建议动作”输出表格。
2. 保留原文中的关键证据。
3. 最后保存为 Markdown 文件。
```

## 常用任务模板

### 整理会议记录

```text
请阅读我上传的会议记录，整理会议结论、待办事项、负责人、截止时间和风险点，输出 Markdown 表格。
```

### 分析表格

```text
请分析工作区中的 sales.csv，找出本周变化最大的 10 个指标，说明可能原因，并生成一份简短报告。
```

### 生成回复草稿

```text
请根据最近一条客户消息起草回复。先给我草稿，不要直接发送。
```

### 生成报告

```text
请基于我上传的材料生成一份 HTML 报告，包含摘要、关键发现、证据和下一步建议。
```

### 先计划后执行

```text
请先列出处理计划，说明需要读取哪些材料、会生成哪些结果。等我确认后再继续执行。
```

## 执行中怎么看

任务提交后，重点看下面几类状态：

| 页面表现 | 代表什么 | 处理方式 |
| --- | --- | --- |
| 直接回复 | 任务已给出结果或阶段结果 | 检查是否符合要求 |
| 请求补充信息 | 材料或目标不够明确 | 在同一会话补充 |
| 工具调用 | 正在读取、生成、查询或发送 | 等待完成或按提示确认 |
| 等待审批 | 动作有风险，需要人工确认 | 看清楚影响后再决定 |
| 执行失败 | 某一步没有完成 | 根据错误提示修正任务 |

不要重复提交完全相同的任务。优先在原会话中补充说明，避免生成重复结果。

## 验收结果

拿到结果后，可以按下面清单检查：

- 是否完成了最初目标。
- 是否使用了指定材料。
- 是否遵守了范围限制。
- 是否输出为要求的格式。
- 是否保存到了工作区。
- 是否有需要人工确认的外部动作。
- 是否缺少证据、来源或关键字段。

如果不满意，继续补充具体修改要求。比如“增加证据列”“删掉推测内容”“按优先级排序”“保存成 CSV”。

## 常见误区

| 写法 | 问题 | 更好的写法 |
| --- | --- | --- |
| 处理一下 | 目标不清楚 | 请总结这份文档的结论和待办 |
| 看看有没有问题 | 验收标准不清楚 | 请找出合同中付款、交付和违约条款的风险 |
| 发给客户 | 风险动作不明确 | 先生成客户回复草稿，确认后再发送 |
| 按之前的来 | 新会话可能没有上下文 | 简要复述要沿用的规则 |
| 全部分析 | 范围过大 | 先分析最近 30 天数据，输出前 10 个异常 |

## 常见问题

### 任务是否必须一次完成？

不必须。复杂任务可以分阶段推进。你可以要求 xAgent 先列计划，确认后再执行。

### 任务是否等同于工具调用？

不是。工具只是完成任务的手段。普通用户不需要指定工具名，只要说明想完成的动作。

### 什么时候需要拆分任务？

当材料很多、步骤复杂、需要等待外部结果或需要多次确认时，建议拆分。

### 可以让 xAgent 自动保存结果吗？

可以。在任务里明确写“保存为 Markdown / CSV / HTML 报告”等格式。
