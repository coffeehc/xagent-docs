# 菜单与功能入口

## 适用对象

本文适合第一次进入 xAgent 控制台，希望先按菜单了解“从哪里进入、能做什么”的用户。

## 工作台

工作台是普通用户最常使用的一组入口。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| 仪表板 | 查看当前运行概况 | 看 token 统计、活跃任务、等待审批、最近会话 |
| [Agent会话](/docs/user-guide/agent-session) | 提交需求和查看执行过程 | 输入任务、上传附件、查看回复、确认工具动作、继续追问 |
| 工作编排 | 开发中的长期流程入口 | 当前不作为主要使用入口，等功能开放后再按页面提示使用 |
| 工作区文件 | 查看文件和产物 | 浏览目录、预览文件、下载报告、查找会话产物 |
| 会话列表 | 管理历史会话 | 搜索会话、进入对话、查看主会话和子会话、删除无用会话 |

建议普通用户从 Agent会话 开始。这里的 Agent 可以简单理解为“工作助手”，不需要理解技术含义。如果任务涉及文件，再进入工作区文件；如果要查历史，再进入会话列表。

## 运行治理

运行治理用于管理执行过程中的风险动作、自动触发、可用能力和外部连接。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| 审批 | 查看需要人工确认的敏感动作 | 查看审批记录、确认或拒绝待处理动作、追踪高风险操作 |
| [触发器管理](/docs/user-guide/trigger) | 让任务定时或由外部事件启动 | 新增触发器、启用或禁用触发器、手动触发一次、查看运行时间 |
| [智能体管理](/docs/user-guide/agent-management) | 管理可选的工作入口 | 新建或查看可选工作入口、选择默认工具和技能、调整展示信息 |
| [Skill 管理](/docs/user-guide/skill) | 管理个人技能 | 查看技能、导入技能、创建技能草稿、提交或启用技能 |
| [我的工具](/docs/user-guide/tool) | 查看当前用户可用工具 | 检查工具是否可用、查看工具来源和说明、调整个人工具设置 |
| 我的 MCP | 管理个人外部能力服务 | 新增服务、测试连接、查看发现到的工具 |
| [我的连接](/docs/user-guide/connector) | 管理个人外部连接 | 查看连接状态、发起授权、绑定外部账号、断开连接 |
| 密钥管理 | 保存个人密钥引用 | 新增密钥、更新密钥、删除不再使用的密钥 |

这里的页面不要求普通用户理解底层协议。只要知道：审批用于风险控制，触发器用于自动提交任务，工具和技能用于扩展能力，连接和密钥用于接入外部系统。

## 个人设置

个人设置只影响当前用户自己的使用体验和个人策略。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| 偏好设置 | 设置默认协作偏好 | 设置回复语言、详细程度、长期规则和默认工作方式 |
| 个人审批策略 | 设置个人风险动作规则 | 调整哪些动作需要确认、哪些动作允许或拒绝 |
| 主题设置 | 调整界面显示 | 切换亮色/暗色、选择配色、调整字体密度 |
| 邮箱管理 | 管理个人邮箱账号 | 添加邮箱、测试发信、选择默认邮箱 |

## 用户管理

用户管理只对管理员显示，用于维护账号和组织身份。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| 用户账号 | 管理本地账号 | 新增用户、重置密码、调整用户角色 |
| 用户组 | 管理用户分组 | 创建分组、维护成员、配合权限策略使用 |
| 企业账号 | 管理企业身份 | 配置企业身份源、外部身份和用户映射 |

## Agent 治理

Agent 治理只对管理员显示，用于维护系统级能力和接入状态。普通用户可以忽略这一组菜单。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| [智能体管理](/docs/user-guide/agent-management) | 管理系统级工作入口 | 新建或沉淀公共配置、选择默认技能和工具 |
| 外接系统 | 查看外部能力状态 | 查看接入系统、能力数量、健康状态、风险等级 |
| [审批策略](/docs/user-guide/approval-policy) | 管理系统级审批规则 | 维护全局审批规则、调整敏感动作策略 |
| [Skill 管理](/docs/user-guide/skill) | 管理公共技能 | 查看全部技能、处理提交、进入技能详情 |
| [Tool 管理](/docs/user-guide/tool) | 管理系统工具 | 查看工具清单、来源、启用状态、风险和说明 |
| [Connector 管理](/docs/user-guide/connector) | 管理外部连接器 | 添加连接器、查看连接状态、刷新状态、删除连接器 |
| MCP 配置 | 管理全局外部能力服务 | 新建服务、测试连接、保存配置、查看工具数量 |
| 系统组件 | 管理系统组件 | 查看文件处理等组件状态和配置 |

## 系统配置

系统配置只对管理员显示，用于维护服务运行所需的基础配置。

| 菜单 | 用途 | 常见操作 |
| --- | --- | --- |
| [模型配置](/docs/user-guide/model-config) | 管理模型和 provider | 新建模型、测试连接、保存模型、调整能力开关 |
| 系统配置 | 管理基础服务配置 | 修改监听地址、系统邮箱、安全密钥和 token 有效期 |
| Agent 角色配置 | 管理内置角色配置 | 查看和调整系统内置角色 |

## 常见问题

### 为什么我的菜单比截图少？

菜单会按账号权限裁剪。普通用户通常不会看到用户管理、Agent 治理和系统配置。

### 页面里看到真实邮箱、token 或内部地址怎么办？

不要把包含敏感信息的截图直接放进文档或公开渠道。截图前应切换到示例环境，或对敏感区域打码。

### 第一次使用应该看哪个菜单？

先进入 Agent会话。需要查看文件时进入工作区文件；需要自动执行时进入触发器管理；需要管理个人能力时进入 Skill 管理、我的工具、我的连接或密钥管理。
