# 记忆

> 状态：实验性能力，接口可能变化。

## 适用对象

本文适合希望 xAgent 跨会话记住偏好、决策或长期背景的用户。

## 这是什么

记忆是面向长期上下文的能力。它不等同于会话历史，也不等同于“把所有聊天都保存下来”。

## 什么时候使用

当某些信息未来还会反复影响任务时，可以考虑让 xAgent 记住，例如：

- 用户偏好的输出语言、格式和风格。
- 项目的长期背景。
- 已确认的关键决策。
- 常用流程或协作规则。

## 基本用法

使用记忆时注意：

- 不要把临时材料、一次性路径或未确认事实当成长期记忆。
- 重要偏好最好明确表达，例如“以后这个项目默认使用中文输出”。
- 如果当前要求和旧记忆冲突，以当前用户明确说明为准。
- 记忆是实验性能力，具体提取、审核和生效策略可能调整。

## 常见问题

### 记忆是否自动保存所有消息？

不会。长期记忆需要有价值、可追溯，并且适合未来复用。

### 记忆是否替代会话历史？

不替代。会话历史用于当前会话上下文，记忆用于长期复用。

### 记忆规则是否已经完全稳定？

还没有完全稳定。使用时应把它视为辅助上下文，而不是唯一事实来源。
