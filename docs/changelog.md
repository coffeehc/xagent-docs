# Changelog

## Unreleased

### Added

- 新增用户向技能、外部连接和触发器使用说明。
- 新增菜单与功能入口说明，并加入控制台真实截图。

### Changed

- 主 Agent 与子 Agent 的对比说明从智能体管理页收口到智能体说明页。
- 智能体说明补充主 Agent 与子 Agent 的差别，并说明复杂任务中子 Agent 由主 Agent 按需拆分使用。
- 模型配置说明补充当前开放属于过渡期，后续将由统一模型管理和任务路由接管。
- 细化使用手册，新增 Agent会话、智能体管理、审批策略和模型配置页面，并扩写任务、工作区、工具、Skill、触发器、外部连接和长任务说明。
- 将文档站定位调整为使用手册优先，同时保留项目介绍入口。
- 将入门路径从命令启动改为功能导览，命令和配置降级到参考信息。
- 将默认导航收敛到开始使用、使用手册、FAQ 和更新记录，技术资料不再作为普通用户主路径。
- 首页增加核心能力和上手路径分区，让新用户先按产品能力理解入口。
- 替换菜单说明中的控制台截图，并补充智能体管理、审批策略等浅色完整页面截图。
- 将工作编排菜单说明调整为开发中入口，避免写成当前主要使用路径。
- 用户手册页面改用中文功能名，技术术语只在必要说明和技术参考中出现。
- 文档维护规则增加必要截图要求。
- 首页和项目介绍补充“多用户智能体门户”的定位，以及管理员、普通用户和进阶用户的使用路径。
- 项目介绍补充免费二进制版本的发布定位和商业边界。
- 补充工作区隔离、MCP/外部连接扩展、技能扩展和会话中模型切换等产品特点说明。
- 重写快速开始、核心概念和用户指南主页面，减少维护者视角占位文案。
- 补充已核对的启动命令、生产构建命令和常用配置字段。
