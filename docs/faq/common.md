# 常见问题

## 适用对象

本文适合所有 xAgent 文档读者。

## 这是什么

本文集中回答第一阶段文档站中的常见问题。

## 什么时候使用

当读者无法判断文档状态、术语边界或文档维护规则时，先查看本文。

## 基本用法

### 为什么很多页面标记为 experimental 或 planned？

因为第一阶段目标是建立官方文档中心，不把未稳定能力写成已实现事实。

### 为什么不写完整 API？

没有代码依据时，不写具体 API、字段和错误码。

### 为什么需要术语表？

xAgent 涉及 Agent、Tool、Skill、Connector、Memory 等多个概念。术语不统一会直接影响文档和实现讨论。

### 文档站是否包含登录、评论或插件市场？

第一阶段不包含。

### Changelog 放在哪里？

Changelog 位于 [docs/changelog](/docs/changelog)。
