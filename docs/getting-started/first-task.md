## 常见问题

### Task 是否一定会生成 SubAgent？

不一定。SubAgent 面向长任务拆分，不应默认写成所有 Task 的固定行为。

### Task 是否等同于一次用户消息？

不等同。一次用户消息可以触发一个 Task，也可能只是补充上下文。

### 当前能否写命令示例？
