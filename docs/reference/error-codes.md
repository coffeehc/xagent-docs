# Error Codes

> 状态：规划中。

## 适用对象

本文适合需要排查 xAgent 错误和维护错误说明的用户。

## 这是什么

Error Codes 用于集中记录稳定错误码、错误含义、触发条件和处理建议。

## 什么时候使用

当错误码已经稳定并且用户需要据此排查问题时，补充本文。

## 基本用法

后续每个错误码应包含：

- 错误码。
- 所属模块。
- 含义。
- 触发条件。
- 用户可执行处理方式。

## 常见问题

### 可以先写常见错误文案吗？

可以写排查方向，但不能编造错误码。

### 错误码是否需要稳定格式？

需要。稳定后应在本文集中维护。

### 错误码变化是否需要 Changelog？

需要。
