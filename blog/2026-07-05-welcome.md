# xAgent 文档站启动

xAgent 需要一个低维护、长期可扩展的官方文档中心，用来承载使用手册、架构说明、扩展开发和版本更新记录。

第一阶段文档站包含 Docs、Blog 和 Changelog：

{/* truncate */}

- Docs 用于稳定说明、概念解释、使用手册和开发指南。
- Blog 用于技术文章、产品更新和社区运营内容。
- Changelog 用于记录用户可见变化和文档版本演进。

当前阶段不会引入登录、评论、多语言、文档版本化、API Playground 或动态后端。文档站优先保持结构清晰、维护成本低，并为 Codex 辅助维护留下明确规则。

后续内容会逐步沉淀使用手册、架构设计、扩展开发和场景实践。所有新增内容都应遵守术语一致性、状态标记和不编造未实现功能的基本规则。
