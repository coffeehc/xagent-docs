# xAgent 文档站启动

xAgent 需要一个低维护、长期可扩展的官方文档中心。这个站点的主线是使用手册：先介绍 xAgent 项目，再帮助用户完成启动、会话、任务、工具、Skill、Connector、Workspace、Memory 和排错。

文档站包含 Docs、Blog 和 Changelog：

{/* truncate */}

- Docs 用于项目介绍、快速开始、用户手册、扩展维护和技术参考。
- Blog 用于技术文章、产品更新和社区运营内容。
- Changelog 用于记录用户可见变化和文档版本演进。

当前文档站不会引入登录、评论、多语言、文档版本化、API Playground 或动态后端。它优先保持结构清晰、维护成本低，并为 Codex 辅助维护留下明确规则。

后续内容应优先补齐用户如何完成任务，再补充扩展开发、技术参考和场景实践。所有新增内容都应遵守术语一致性、状态标记和不编造未实现功能的基本规则。
